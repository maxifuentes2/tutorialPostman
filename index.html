Gestión de Categorías

POST Nueva Categoría
/api/categories 
{ "name": "Jujutsu Kaisen" }
PUT Editar Nombre
/api/categories/{id} 
{ "name": "Cine & Series" }
GET Ver todas las IDs
/api/admin/categories
DELETE Eliminar (Forzado)
/api/categories/{id}

💡 Recordatorios Técnicos

  • • Las rutas con /{id} requieren el número real de la base de datos
  • • En Postman: Body -> raw -> JSON
  • • Galería: Es un array [ "url1", "url2" ]
